# Welcome to Nextra

This is your documentation homepage. Start editing this file to add your content!

## Getting Started

You can create more pages by adding `.mdx` files in the `app` directory.

### Features

- **MDX Support**: Write content with Markdown and embed React components
- **Built-in Search**: Fast search across all your documentation
- **Dark Mode**: Automatic dark mode support
- **Mobile Friendly**: Responsive design out of the box

## Next Steps

1. Edit this file (`app/page.mdx`) to customize your homepage
2. Create new pages by adding more `.mdx` files
3. Configure the navigation in `app/_meta.ts` or `app/_meta.tsx`
4. Customize the theme in `app/layout.tsx`
